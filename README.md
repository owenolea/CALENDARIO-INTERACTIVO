## 📅 Descripción del Proyecto
Calendario interactivo desarrollado con tecnologías web  (HTML5, CSS3, JavaScript). El sistema permite visualizar un calendario mensual con navegación entre meses, destacando el día actual y los feriados argentinos. Incluye funcionalidades de persistencia local para recordar el nombre del usuario y preferencias de modo visual, además de un boton para cambiar entre modo infantil (colores vibrantes y animaciones) y modo argentino (estilo profesional).

Este proyecto resuelve la necesidad de una herramienta simple y accesible para la visualización de fechas importantes, dirigido a usuarios que requieren un calendario básico con personalización y recordatorios visuales.

## 🛠️ Stack Tecnológico

- **Almacenamiento**: localStorage (navegador)

## 🏗️ Arquitectura del Sistema
Este proyecto sigue una arquitectura frontend :

- **Capa de Presentación**: HTML5 semántico, CSS3 con transiciones y animaciones, JavaScript para interactividad.
- **Persistencia**: localStorage del navegador para datos del usuario.
- **Interfaz**: Diseño responsivo con modos visuales adaptativos.
